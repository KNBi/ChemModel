1. For which elements does the platform works?

- For now application works for:
H, C, N, O, P, Na, K, Br, I, F, Cl, S, Li

2. Does the solution work for both organic and inorganic substances?

- No, for now solution works only for organic substances.

3. Does the solution return all possible results?

- No, solutions follows some rules when calculating results.

4. Which rules?

- carbon atom doesn't bond with nitrogen atom by oxygen atom
- sulfur atom doesn't bond with nitrogen atom
- oxygen, nitrogen and sulfur atoms don't bond with halogens
